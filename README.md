# 🚀 AETHER – Modern Digital Landing Page

AETHER is a modern, visually striking landing page built using **HTML, CSS, and JavaScript**.  
It features gradient UI elements, glowing effects, and a clean futuristic design.

---

## ✨ Features

- 🎨 Gradient text and glowing UI
- 💡 Glassmorphism header
- 🔵 Animated glow indicators
- 📱 Responsive design
- ⚡ Smooth and modern layout

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Gradients, Animations)
- JavaScript (for interactivity)
